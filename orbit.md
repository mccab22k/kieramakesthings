# Orbit, A Fancy Planner

App Store: https://apps.apple.com/us/app/orbit-a-fancy-planner/id6776764489

Web app: https://mccab22k.github.io/orbit-a-fancy-planner/

## Summary

Orbit is a visual planner built around a simple idea: your day is a clock, not a list.

Instead of stacking events vertically, Orbit displays a schedule as a 24-hour circular dial. The goal is to make time, routines, and energy easier to understand at a glance.

## Core Features

- 24-hour circular planner
- two concentric planning rings
- drag-and-drop scheduling
- edge drag to resize blocks
- recurring tasks
- completion-based task regeneration
- task inbox
- reusable templates
- day, week/wave, and month-style planning views
- local-first storage

## Use Case

Orbit is designed for visual planning, routine-building, and reducing the friction of managing time. It is especially useful when a list or calendar view feels too abstract.
