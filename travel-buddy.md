# Travel Buddy

Status: work in progress.

## Summary

Travel Buddy is an early travel planning project. The goal is to make it easier to collect trip ideas, compare destinations, organize itinerary notes, and keep simple logistics in one place.

## Possible Features

- destination cards
- shared preference notes
- itinerary outline
- packing checklist
- budget notes
- links for maps, lodging, food, and activities

## Next Steps

- define the main use case
- sketch the first screen
- choose a minimal data model
- decide whether the first version is local-only or shared
